## 🎯 Sobre o projeto

Este site foi criado como um **projeto conceitual** para uma academia real de Macapá/AP, que possui nota 4,9 no Google Maps com 31 avaliações. O objetivo era transformar a presença digital da academia, tornando-a encontrável no Google e gerando mais matrículas via WhatsApp.

**Por que este projeto é relevante?**
- 52% das pequenas empresas brasileiras ainda não possuem site
- Academias locais perdem clientes para concorrentes que estão online
- Um site institucional gera credibilidade e autoridade no mercado

## ✨ Funcionalidades

- ✅ **Design 100% responsivo** (funciona em celular, tablet e desktop)
- ✅ **Selo de avaliação** 4,9★ com 31 reviews (dados reais do Google)
- ✅ **Botão WhatsApp flutuante** com mensagem automática
- ✅ **Depoimentos reais** de clientes da academia
- ✅ **Mapa interativo** com localização exata
- ✅ **Cards de diferenciais** (climatização, equipamentos novos, profissionais)
- ✅ **Tabela de planos** com chamada para ação
- ✅ **Integração com Instagram** @biosyncfitness
- ✅ **Menu hambúrguer** para mobile
- ✅ **Otimizado para SEO** (meta tags, título, descrição)
